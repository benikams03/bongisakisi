## Description
BongisaKisi est une application desktop développée avec Electron permettant de gérer efficacement les activités liées à une pharmacie (stocks, ventes, produits, etc.).
L'objectif est de digitaliser et simplifier la gestion quotidienne avec une interface moderne et intuitive.
